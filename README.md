# Carmen's Garden Cafe POS

Carmen's Garden Cafe - Offline-First Flutter POS System.

> [!IMPORTANT]
> This project is custom built only for the device **"OUKITEL WP18"** a pos system for Carmens Garden Cafe.

## Tech Stack

- **Framework**: [Flutter](https://flutter.dev) (SDK ^3.5.4)
- **State Management**: [Riverpod](https://riverpod.dev)
- **Navigation**: [Go Router](https://pub.dev/packages/go_router)
- **Database (Local)**: [sqflite](https://pub.dev/packages/sqflite) (Offline-first approach)
- **Database (Cloud/Sync)**: [Supabase](https://supabase.com)
- **Networking**: [Dio](https://pub.dev/packages/dio)
- **Code Generation**: Freezed & JSON Serializable
- **UI Components**: fl_chart (for reporting)

## Getting Started

This project is specialized for the OUKITEL WP18 hardware. Ensure the device environment is correctly configured for Flutter deployment.
